One-time Google Cloud setup

  2. APIs & Services → Library → search for and enable Google Photos Picker API
  3. APIs & Services → OAuth consent screen → set up consent screen (External, add your email as test user)
  4. Credentials → Create Credentials → OAuth client ID → type: Web application
  5. Under Authorised redirect URIs add exactly:
