One-time Google Cloud setup

  1. Go to console.cloud.google.com → create or select a project
  2. APIs & Services → Library → enable Google Photos Picker API
  3. APIs & Services → OAuth consent screen → External → add your email as a test user
  4. Credentials → Create Credentials → OAuth client ID → type: Web application
  5. Under Authorised redirect URIs add: Copy
  6. Copy your Client ID and Client Secret into the fields below
